Example: "hello world" → "HELLO WORLD" (UPPERCASE)

"HELLO WORLD" → "hello world" (lowercase)

"hello world" → "Hello World" (Title Case)

Types of Text Case Conversion

🔠 UPPERCASE

Converts all letters to capitals. Example: "hello" → "HELLO"

🔡 lowercase

Converts all letters to small letters. Example: "HELLO" → "hello"

📘 Title Case

Capitalizes first letter of every word. Example: "hello world" → "Hello World"

📖 Sentence Case

Capitalizes first letter of each sentence. Example: "hello world." → "Hello world."

📌 Capitalize Each Word

Capitalizes first letter of every word. Similar to title case
