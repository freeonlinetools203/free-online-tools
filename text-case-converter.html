Types of Text Case Conversion

1. Uppercase Converter

The uppercase converter transforms all letters in your text into capital letters. This format is commonly used for headings, titles, and emphasis in documents.

Example:

hello world → HELLO WORLD

2. Lowercase Converter

The lowercase converter converts all letters into small letters. This is useful when text is accidentally typed in uppercase.

Example:

HELLO WORLD → hello world

3. Title Case Converter

The title case converter capitalizes the first letter of every word. This format is widely used for headings, blog titles, and article titles.

Example:

hello world → Hello World

4. Sentence Case Converter

The sentence case converter capitalizes only the first letter of a sentence. This format is commonly used in normal writing and paragraphs.

Example:

hello world. this is a tool → Hello world. This is a tool
