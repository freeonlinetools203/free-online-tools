What Makes a Password Secure?

A secure password should:

  • Be at least 12–16 characters long
  • Include uppercase and lowercase letters
  • Contain numbers
  • Include special symbols
  • Avoid dictionary words
  • Not include personal information

Example of weak password: password2024

Example of strong password: X9#tL2!qP7@vZ4

The difference is clear — complex and random passwords provide significantly better protection.

Why You Should Use Unique Passwords

Using the same password across multiple websites is extremely risky. If one website experiences a data breach, hackers may try that same password on your other accounts.

Best practice:

  • Use a unique password for every account.
  • Store passwords in a password manager.
  • Enable Two-Factor Authentication (2FA).
